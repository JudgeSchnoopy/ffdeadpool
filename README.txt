This project is under development by Jeff Golden.

Alpha by HTML5 UP has been used as a website template.  The template is free.

ffdeadpool (Fantasy Football Deadpool) is a fantasy football game where points
are awarded based on how many games your athlete misses.
Yes, this is a little morbid and cynical, but we're just trying
to have some fun.

The athletes will or will not get hurt regardless of our little game.

This site uses HTML, CSS, and PHP.  The backend is MySQL.
